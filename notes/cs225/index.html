C++ Classes

Rule of Three

If you define any of destructor, copy constructor, or copy assignment — define all three.

class MyClass {
public:
  MyClass();                          // default ctor
  MyClass(const MyClass& other);      // copy ctor
  MyClass& operator=(const MyClass&); // copy assign
  ~MyClass();                         // destructor
};

Recursion

Every recursive function needs a base case and progress toward it. For maze solving, the base case is reaching the exit or a dead end.
