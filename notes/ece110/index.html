Thévenin & Norton Equivalents

Key Idea

Any linear circuit with sources and resistors can be reduced to a single voltage source V_th in series with a resistor R_th (Thévenin), or a current source I_N in parallel with R_N (Norton).

Steps

  • Remove the load. Find V_oc (open-circuit voltage) → this is V_th.
  • Kill all independent sources (voltage → short, current → open). Find R_eq from terminals → R_th.
  • Norton: I_N = V_th / R_th.

MOSFETs

nMOS Operating Regions

Three regions based on V_GS and V_DS:

  • Cutoff: V_GS < V_tn — device is off, I_D = 0.
  • Triode/Linear: V_DS < V_GS − V_tn — acts like a resistor.
  • Saturation: V_DS ≥ V_GS − V_tn — current source behavior.
I_D (sat) = (k_n / 2)(V_GS - V_tn)²

Nodal Analysis

KCL at each node

Assign a reference node (ground). Write KCL at each unknown node: sum of currents leaving = 0. Express all currents in terms of node voltages.

Damped Oscillators

Second-order RLC

Series RLC circuit satisfies:

L·(d²i/dt²) + R·(di/dt) + (1/C)·i = 0

Damping ratio ζ = R/(2)·√(C/L). Under-damped when ζ < 1 → oscillatory response.
